# Secure Note Vault
Encrypted notes CLI app using AES (Fernet) and PBKDF2 key derivation.

## Features
- AES encryption with password
- Typer CLI
- Secure vault file storage
- Fully ready for GitHub portfolio

## Run
```
call .venv\Scripts\activate
secure-note init
secure-note add --title "Email Password" --body "abc123"
secure-note list
```
